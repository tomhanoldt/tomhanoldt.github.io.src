# HTML-Youtube-Mixer

The simple problem of having many people in one room starting youtube videos...
- make it a bit more ear friendly
- mix two youtube videos and switch between them with a volume fader
- download from [github](https://github.com/tomhanoldt/html-youtube-mixer) and open index.html with a browser
- a problem is that not all media is available for extern youtube apps, there was the idea to use a the java-fx browser component to load the media in a "real" browser
