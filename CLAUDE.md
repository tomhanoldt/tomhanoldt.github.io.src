# Claude Code project instructions

See [`AGENT.md`](AGENT.md) for the actual project context, commands, and
conventions - that file is the canonical, tool-agnostic source so it stays
useful for any agent working in this repo, not just Claude Code.

The two Claude-Code-specific things worth repeating here, since they come
up often: **never commit without being explicitly asked**, even after
finishing a chunk of work, and **always run `make lint` and `make test`
clean** before any commit that does happen. Both are stated in `AGENT.md`
too.
